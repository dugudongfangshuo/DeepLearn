# 🛎项目说明

- 该项目包含自然语言处理，计算机视觉，语音处理的相关代码。

- 主要对代码进行收录总结，将算法进行图示说明，代码进行注释，并配有代码的逻辑关系讲解。

- 只收录如下特点的代码：使用语言只有python3，可以在windows下运行的。

- 项目内每个算法都放到单独的一个文件夹下，各个算法相互独立运行，不相互依赖和影响，每个算法都会配有python依赖说明，可以直接创建虚拟环境后安装依赖快速运行代码。
